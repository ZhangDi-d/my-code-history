#### ssh命令

ssh 连接Node

```shell
ssh username@ip
```

退出连接

```shell
ctrl + d
```

不会小心按了上面的命令，重新连接

```shell
reconnect
```



#### 查看从本机访问其他网址的路由

linux:

```shell
traceroute ingress
```

windows：

```powershell
tracert ingress
```



#### scp 命令

- 使用root账号密码

```shell
scp 文件名 目标ip:目标目录
```

- 指定账号，使用秘钥

```shell
scp -i 秘钥文件 传输的文件 用户名@目标ip：目标目录
```
