#### ssh命令

ssh 连接Node

```shell
ssh username@ip
```

退出连接

```shell
ctrl + d
```

不会小心按了上面的命令，重新连接

```shell
reconnect
```



#### 查看从本机访问其他网址的路由

linux:

```shell
traceroute ingress
```

windows：

```powershell
tracert ingress
```



#### scp 命令

- 使用root账号密码

```shell
scp 文件名 目标ip:目标目录
```

- 指定账号，使用秘钥

```shell
scp -i 秘钥文件 传输的文件 用户名@目标ip：目标目录
```



#### awk 命令

- 打印查到内容的第一列

```shell
| awk '{print $1}'
```

#### sed 命令

- 替换

```shell
sed 's/src/target/g'
```



#### 清楚当前回话执行的命令

```shell
history -r
```



#### 查看进程的端口号

```shell
netstat -nap | grep xxx
```
